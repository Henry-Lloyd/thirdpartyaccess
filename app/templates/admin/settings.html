Download SQLite Backup

Raw database file, drop-in replacement

Import Data

Restore data from a previously exported JSON file. This uses merge mode — existing records are updated, new records are added, and nothing is deleted. A backup is automatically created before importing.

Data Persistence Tips:

  • Export regularly (especially before deploying updates or migrating hosts).
  • The JSON export contains all tables: users, passwords, providers, payments, payouts, reviews, notifications, settings, and more.
  • The SQLite backup is a direct copy of the database file — you can replace the data/database.sqlite file on a new deployment.
  • Importing uses merge mode: existing rows are updated, new rows are added. No data is deleted.
  • A safety backup is automatically created before every import.
